**题目链接**：https://leetcode-cn.com/problems/split-a-string-in-balanced-strings/

**解题思路**：

设置一个标记，遇到R就+1，遇到L就减1，n等于0，res就加一。贪心的思想，保证分割的平衡字符串尽可能多。

**时间复杂度**：$O(N)$。N是字符串str的长度。

**空间复杂度**：$O(1)$。

```python
class Solution:
    def balancedStringSplit(self, s: str) -> int:
        n = 0
        res = 0
        for c in s:
            if c == 'R':
                n += 1
            else:
                n -= 1
            if n == 0:
                res += 1
        return res
```
